Адмінка товарів
Це редактор каталогу для сайту, який не має власного сервера — тому зміни зберігаються у файл products.json, а не «на льоту». Порядок такий: 1) редагуйте товари тут як завгодно (додавайте, змінюйте ціни й наявність); 2) натисніть «Зберегти products.json» — завантажиться оновлений файл; 3) замініть ним файл products.json там, де розміщений сайт (хостинг / Netlify / GitHub) — і зміни з'являться для всіх відвідувачів. Поки не замінили файл на сервері, чернетка зберігається лише у цьому браузері.
